Uji coba 2



Kesimpulan: Ketika atribut "name" tidak diisi, semua pilihan radio button bisa dipilih sekaligus. Ketika atribut "name" diisi dengan nilai yang SAMA, hanya satu pilihan yang bisa dipilih (saling eksklusif). Ketika atribut "name" diisi dengan nilai yang BERBEDA, semua pilihan bisa dipilih sekaligus karena dianggap kelompok berbeda.

Uji coba 3



Kesimpulan: Checkbox berbeda dengan radio button. Pada checkbox, pengguna bisa memilih lebih dari satu pilihan sekaligus. Atribut "name" pada checkbox digunakan untuk mengelompokkan data saat dikirim ke server, bukan untuk membatasi pilihan.

Uji coba 4




Kesimpulan: Input type="color" menampilkan color picker untuk memilih warna. Input type="date" menampilkan kalender untuk memilih tanggal. Input type="button" dengan atribut onclick dapat menjalankan fungsi JavaScript ketika diklik, seperti menampilkan alert box.

Uji coba 5




Kesimpulan: Input type="datetime-local" memungkinkan pemilihan tanggal dan waktu sekaligus. Input type="month" memungkinkan pemilihan bulan dan tahun. Input type="email" secara otomatis memvalidasi format email yang dimasukkan. Tampilan input ini bisa berbeda-beda tergantung browser yang digunakan.

Uji coba 6




Kesimpulan: Input type="image" menampilkan gambar sebagai tombol submit. Input type="file" digunakan untuk memilih file dari komputer pengguna. Input type="hidden" tidak terlihat oleh pengguna tetapi datanya tetap dikirim ke server, berguna untuk menyimpan data tersembunyi seperti ID pengguna.

Uji coba 7



Kesimpulan: Input type="number" menampilkan field angka dengan tombol naik/turun, atribut min dan max membatasi nilai yang bisa dimasukkan. Ketika min diubah menjadi -50, nilai minimum yang bisa dimasukkan menjadi -50. Input type="range" menampilkan slider, nilai min dan max menentukan batas geser slider.

Uji coba 8




Kesimpulan: Input type="url" memvalidasi format URL yang dimasukkan. Input type="search" mirip text biasa namun biasanya memiliki tombol hapus (x). Input type="tel" digunakan untuk nomor telepon, atribut pattern memvalidasi format nomor. Tampilan dan validasi bisa berbeda tergantung browser yang digunakan.

Uji coba 9



Kesimpulan: Input type="time" menampilkan picker untuk memilih jam dan menit. Input type="week" menampilkan picker untuk memilih minggu dalam satu tahun. Kedua tipe input ini memiliki tampilan yang berbeda di setiap browser, beberapa browser mungkin tidak mendukung tipe input ini sepenuhnya.
