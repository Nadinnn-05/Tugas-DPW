Jika Anda mengklik tombol "Kirim", data formulir akan dikirim ke halaman bernama "/action_page.php".

Kesimpulan :
1. Tombol Reset : mengosongkan semua isian kembali ke nilai awal (value).
2. Tombol Kirim : mengirimkan data form ke halaman yang ditentukan di atribut action.
3. target="_self" : halaman terbuka di tab yang sama.
   target="_blank" : halaman terbuka di tab baru.
   target="_parent" : halaman terbuka di frame induk.
   target="_top" : halaman terbuka penuh di jendela saat ini.
4. method="get" : data dikirim lewat URL (terlihat di address bar), cocok untuk data tidak sensitif.
   method="post" : data dikirim lewat body HTTP (tidak terlihat di URL), lebih aman untuk data sensitif.
