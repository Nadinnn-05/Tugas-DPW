2. Rectangle (Persegi Panjang Biru)



3. Rectangle (Sudut Bulat + Transparan)



4. Polygon (Segitiga Hijau)


Kesimpulan:
- <circle>: membuat lingkaran. cx & cy = titik pusat, r = jari-jari.
- <rect>: membuat persegi panjang. rx & ry untuk sudut membulat.
- <polygon>: membuat poligon dari kumpulan titik koordinat (points).
- opacity: transparansi bentuk (0 = transparan penuh, 1 = solid).
- SVG bersifat vektor sehingga gambar tidak pecah saat diperbesar.
