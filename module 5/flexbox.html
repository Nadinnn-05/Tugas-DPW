Kesimpulan skenario 1: display: flex membuat elemen anak (child) tersusun secara horizontal (baris) secara otomatis. Tanpa flex, elemen div akan tersusun secara vertikal (satu per satu ke bawah).

Kesimpulan skenario 3: flex-direction mengatur arah susunan item flex.
- row = kiri ke kanan (default)
- column = atas ke bawah
- row-reverse = kanan ke kiri
- column-reverse = bawah ke atas

Kesimpulan skenario 4: flex-wrap mengatur apakah item flex boleh pindah ke baris baru.
- nowrap = semua dalam 1 baris meskipun melebihi lebar
- wrap = item turun ke baris berikutnya jika tidak cukup ruang
- wrap-reverse = item naik ke baris sebelumnya

Kesimpulan skenario 5: flex-flow adalah shorthand (singkatan) yang menggabungkan flex-direction dan flex-wrap dalam satu properti.
Contoh: flex-flow: row wrap = arah horizontal + item boleh turun ke baris baru.

Kesimpulan skenario 6: justify-content mengatur posisi/distribusi item secara horizontal (sumbu utama).
- flex-start = rata kiri
- flex-end = rata kanan
- center = tengah
- space-between = jarak rata antar item, tanpa tepi
- space-around = jarak rata antar item, termasuk tepi
