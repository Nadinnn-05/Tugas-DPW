Kesimpulan:
- display: grid: mengaktifkan layout grid pada container.
- grid-template-areas: mendefinisikan nama area untuk setiap bagian layout (header, menu, main, right, footer).
- grid-template-columns: 1fr 2fr 1fr: membagi lebar menjadi 3 kolom — kiri 1 bagian, tengah 2 bagian, kanan 1 bagian.
- grid-template-rows: mengatur tinggi setiap baris.
- grid-area: menentukan item berada di area mana sesuai nama yang sudah didefinisikan di grid-template-areas.
- gap: jarak antar area grid.

CSS Grid sangat cocok untuk membuat layout halaman web yang kompleks karena bisa mengatur posisi elemen secara 2 dimensi (baris dan kolom) sekaligus dengan mudah.
